Cascading Style Sheets

Elementos

Selector(elemento HTML) y declaration block
<style>
h1 {color:XXX; font-size:XXXXpx; text-align:XXXX; background-color:XXXX; margin-right:XXXXpx} -> ESPACIOS
</style>
/* Comentario */

Selectores

ID
"#" para elementos especificos
#p1 {}

CLASS
"." para una clase especifica
.center {}
<p class= "center large">

Grouping

"," para separar los elementos
h1, h2, p {}

Insert CSS

External Style Sheet -> sin tags HTML
<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

Internal Style sheet
<head>
<style>
body, p {...}
</style>
</head>

Inline Styles
<h1 style="color:XXXX;margin-left:XXXXpx;"> </h1>

Cascading Order
1. Inline Style
2. External and Internal style sheets
3. Browser default

Color

< h1 style="border:XXXXpx solid XXXX;">
< h1 style="background-color:rgb(XX,XX,XX);"> -> rgb(0,255)

Backgrounds

Propiedades: color, image, repeat, attachment, position.
h1 {background-color: XXXX}
h1 {background-image: url("image.gif")}
h1 {background-repeat: repeat-x;} -> Horizontal; -y Vertical
h1 {background-repeat:no-repeat;}
h1 {background-position: right top;}
h1 {background-attachment: fixed;}
h1 {background: #ffffff url(image.gif) no-repeat right top;}

Border

Styles: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden, mix
p {border-style: XXXX;}
Width: in, px, pt, cm, em, thin, medium, thick
p {border-width: XXXXpx;}
p {border-color: XXXX;}
p {border-top-style:XXXX; border-right-style:XXXX; border-bottom-style:XXXX; border-left-style:XXXX;}

Shorthand
p {border: 5px solid red;}
p {border-left: ...}
p {border-bottom: ...}
p {border-radius: XXXXpx}

Margins

Styles: -top, -right, -bottom, -left -> En ese orden
Values: auto (center); length in px, pt, cm; length in %; inherit
p {margin: 25px 50px 100px 125px}

Padding

Styles: -top, -right, -bottom, -left -> En ese orden
Values: length in px, pt, cm; length in %; inherit -> No negativos
p {padding: 20px 100px 30px 75px;}
p {width: XXXXpx; padding:XXXXpx; box-sizing: border-box;}

Height & Width

p {heigth:XXXpx; width: XXXX%}
max-width, max-height
