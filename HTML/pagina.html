Introducción





HTML es un lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

!DOCTYPE html = HTML5

<head> = meta information
<title> = titulo del documento
<body> = parte visible del documento
<h1> = encabezado largo
<p> = parrafo
